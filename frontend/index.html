Tentang FilmKu

FilmKu adalah platform informasi film terbaru yang menyediakan data lengkap tentang film-film yang sedang tayang di bioskop dan akan datang.

Website ini dibuat menggunakan HTML, CSS, dan JavaScript dengan integrasi API dari The Movie Database (TMDB) dan API lokal untuk manajemen data film.

Dibangun sebagai proyek untuk memenuhi tugas pembuatan website dengan fitur:

  • Halaman Utama dengan tampilan sederhana menggunakan HTML, CSS, dan JavaScript
  • API sederhana untuk manajemen data film
  • Integrasi API publik (TMDB) untuk data film yang lebih lengkap
  • Testing API dengan Postman
